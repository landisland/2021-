| 《黑客与画家》-  一个东西，一个应用，如果能够位用户创造价值，那么它本身就有价值。如果这个东西是你创造的，那么你就能够赚钱。金钱不是财富，我们其实想要的不是钱，而是钱可以换来的一切东西，服务，商品等等。这些才是真正的财富，而钱只是中介而已。而大部分生意的目的就是为了创造财富，他们做出了人们真正需要的东西。          《文明、现代化、价值投资与中国》 -  价值投资的基本概念只有四个。第一，股票是对公司的部分所有权，而不仅仅是可以买卖的一张纸。第二，安全边际。投资的本质是对未来进行预测，而我们对未来无法精确预测，只能得到一个概率，所以需要预留安全边际。第三，市场先生。市场的存在是为了来服务你的，不是来指导你的。第四，能力圈。投资人需要通过长期的学习建立一个属于自己的能力圈，然后在能力圈范围之内去作投资。           关于投资理念/策略，有知有行整理了ETF拯救世界的一些文章和理念合集，有许多东西常看常新。所以投资的过程也是为了让自己变得更好，当你知行合一的时候，其实投资的结果自然也不会太差了。 |
